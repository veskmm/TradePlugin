# TradePlugin

**TradePlugin** — Minecraft-плагин для безопасной торговли между игроками через GUI-меню.

Плагин позволяет одному игроку отправить приглашение на обмен, второму — принять его, после чего открывается общее меню торговли. Игроки могут выкладывать предметы, видеть предметы друг друга и подтверждать сделку.

---

## Возможности

- Приглашение игрока на торговлю через команду
- Принятие приглашения через команду
- GUI-меню торговли на двух игроков
- Отдельная зона предметов игрока и зона предметов партнёра
- Отображение статуса принятия сделки
- Сброс подтверждения при изменении предметов
- Возврат предметов при отмене сделки
- Завершение сделки только после подтверждения обоими игроками
- Защита от неподдерживаемых действий:
  - Shift-click
  - Move to other inventory
  - Hotbar swap
  - Collect to cursor
- Обработка полного инвентаря (предметы падают на землю)
- Очистка listener-ов после завершения сделки

---

## Команды

### Приглашение

```
/trade invite <ник>
/tr invite <ник>
```

### Принятие

```
/trade accept <ник>
/tr accept <ник>
```

---

## Пример использования

1. Игрок A пишет:
   ```
   /trade invite PlayerB
   ```

2. Игрок B принимает:
   ```
   /trade accept PlayerA
   ```

3. Открывается меню торговли

4. Игроки кладут предметы

5. После изменения предметов подтверждение сбрасывается

6. После подтверждения обоими игроками происходит обмен

## Безопасность

### Закрытие меню
Сделка отменяется, предметы возвращаются

### Изменение предметов
Сбрасывает подтверждение

### Полный инвентарь
Предметы падают на землю

### Защита кликов
Блокируются:
- Shift-click  
- Hotbar swap  
- Move to inventory  
- Collect to cursor  

---

## Ограничения

- Нет таймера приглашения  
- Нет проверки на несколько сделок  
- Нет PlayerQuitEvent  
- Нет config.yml  
- Нет permissions
